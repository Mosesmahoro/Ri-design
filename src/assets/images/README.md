# Image Folder Structure Guide

Place your images in the following folder structure:

## Logo
- **Location:** `src/assets/images/logo/`
- **File:** `logo.png` or `logo.svg`
- **Usage:** Used in the header and footer

## Portfolio / Work Images
- **Location:** `src/assets/images/portfolio/`
- **Recommended files:**
  - `refugee-2024.jpg` - World Refugee Day Campaign
  - `poster-grow-business.jpg` - Ri Designs Brand Promo
  - `poster-creative.jpg` - Creative Designer Poster
  - `mothers-day.jpg` - Mother's Day Greeting
  - `work-web.jpg` - E-Commerce Platform (Web Development)
  - `work-mobile.jpg` - Fintech App (Mobile App)
  - `work-software.jpg` - School Management System (Software)

## Hero Background
- **Location:** `src/assets/images/hero/`
- **File:** `hero-bg.jpg`
- **Usage:** Hero section background image on homepage

## How to Add Images

1. Create the appropriate folder if it doesn't exist
2. Copy your image files into the corresponding folder
3. The application will automatically reference these images

## Image Recommendations

- **Logo:** PNG with transparent background, min 200x200px
- **Portfolio images:** JPEG, min 1024x1024px for good quality
- **Hero background:** JPEG, min 1920x1080px for full-width display

## Example File Structure
```
src/assets/images/
├── logo/
│   └── logo.png
├── portfolio/
│   ├── refugee-2024.jpg
│   ├── poster-grow-business.jpg
│   ├── poster-creative.jpg
│   ├── mothers-day.jpg
│   ├── work-web.jpg
│   ├── work-mobile.jpg
│   └── work-software.jpg
└── hero/
    └── hero-bg.jpg
```

After adding your images, the website will automatically display them in their respective locations.
